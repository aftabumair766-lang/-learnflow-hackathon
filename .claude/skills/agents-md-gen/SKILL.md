# agents-md-gen

Generate a production-ready AGENTS.md file for any software project.

## Purpose
This skill analyzes a repository and produces a clear, structured AGENTS.md
file describing AI agents, their responsibilities, boundaries, and usage rules.

## When to Use
- When a repository does not have an AGENTS.md file
- When onboarding AI agents into a new or existing project
- When standardizing agent usage across teams

## Inputs
- Repository structure
- Existing documentation (README, docs)
- Project type and tech stack

## Output
- A complete AGENTS.md file tailored to the repository
